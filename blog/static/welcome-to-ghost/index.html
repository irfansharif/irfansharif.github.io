Subheadings

  • Item number one
  • Item number two
    • A nested item
  • A final item

or with numbers!

  1. Remember to buy some milk
  2. Drink the milk
  3. Tweet that I remembered to buy the milk, and drank it

Working with Code

Write inline <code> blocks really easily with back ticks. 4 spaces of indentation for something more comprehensive:

.awesome-thing {
    display: block;
    width: 100%;
}

Ready for a Break?

Throw 3 or more dashes down on any new line and you've got yourself a fancy new divider

Advanced Usage

you can write plain old HTML and it'll still work! Very flexible.

Syntax highlighting now works:

int = 42;  
double = 42.0;  
std::string hello = "Hello World!";

$LaTeX$ supported:

$$\sum_{n=1}^{\infty} \frac{1}{n^2} = \frac{\pi}{6}$$
